代码整体思路：
	这个项目的代码是在正点原子ov2640项目的基础之上进行改动的。我增加了摄像头采集的数据上传至电脑进行显示，同时将各个传感器采集的数据一并在最后传送。这样做传输的数据齐全，但是摄像头帧率极低。另外，我修改了项目的按键内容，key1用于拍照，格式为jpej，存入sd卡（默认240*320，符合2.8寸显示屏）。key0用于调节上传照片的尺寸（默认640*480）。
	先初始化各个引脚，sd卡，屏幕，LED和扬声器，然后判断各个模块是否初始化成功。sd卡未初始化成功则无法拍照，但不影响使用。然后使用死循环用于程序的重复执行。在死循环中，通过使用帧采集完成同步信号为标志的循环，用来传输数据，拍摄照片。每传输一帧照片，传输一次传感器数据。
电导率传感器，TDS值传感器，PH值传感器采用ADC+DMA多路采集的模式进行采集转运；温度传感器使用DS18B20模块进行采集（emmm，这个不太懂）。
	项目文件夹介绍：USER里面存放是主函数，HARDWARE文件夹里面存放是传感器使用的硬件模块，以及字库；system文件夹里面存放的是单片机使用的delay.c  sys.c  usart.c函数。后面带1的是从其他项目移植过去的代码。作用相差不大，但是有些东西不太一样，就没改。ADC文件夹里面存放的是adc以及data_processing函数。ADC函数用于实现多个传感器同时采集以及数据运输，data_processing用于数据处理。ADC函数以及data_processing函数是在店铺给的例程中，我自己改的。
